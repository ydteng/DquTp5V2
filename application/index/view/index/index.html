接口手册

1.公共查询类

  • 获取banner（方法：get）

https://api.brainholepi.com/api/v1/banner

参数值
字段 释义
无 无
返回值
字段 释义
id banner的id号
url 图片路径

  • 获取省份（方法：get）

https://api.brainholepi.com/api/v1/province

参数值
字段 释义
无 无
返回值
字段 释义
id 省份id
name 省份名字

  • 获取学校（方法：get）

https://api.brainholepi.com/api/v1/school/:id

说明：根据省份id获取对应省份的学校

参数值
字段 释义
id 省份id
返回值
字段 释义
id 学校id
name 学校名称

provinceId

 学校所属省份id
level 学校等级
city 学校所在城市

 

 

2.登录注册类

  • 获取Token（方法：post）

https://api.brainholepi.com/api/v1/token/user

说明：code参数放在body中传递

参数值
字段 释义
code 微信小程序code
返回值
字段 释义
token 服务器生成的令牌

  • Token合法性验证（方法：post）

https://api.brainholepi.com/api/v1/token/verify

参数值
字段 释义
token 需要验证的token
返回值
字段 释义
isValid 结果（true或者是false）

 

 

3.用户信息类

  • 添加个人信息及收货地址（方法：post）

https://api.brainholepi.com/api/v1/address

说明：token在header中传入，其他在body中传入，json格式。

参数值
字段 释义
token 令牌（在header中传入）
nickname 昵称（区别于微信名）
real_name 真实姓名
mobile  手机号码
province 省份
school 学校
 detail 详细地址
返回值
字段 释义
code http状态码
msg 成功返回“ok”（主要）
errorCode 错误码
